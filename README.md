# 🏦 Sistema de Atendimento Bancário

## 📌 Descrição

Este sistema foi desenvolvido em linguagem **C** com o objetivo de simular o funcionamento de um **atendimento bancário baseado em controle de senhas**.

Além disso, um dos objetivos do projeto foi **treinar e desenvolver a lógica de programação**, aplicando conceitos fundamentais na construção de um sistema funcional.

---

## ⚙️ Funcionalidades

### 🎟️ Geração de Senhas
Cria senhas específicas de acordo com o tipo de operação bancária desejada.

### 👨‍💼 Atendimento ao Cliente
Realiza a chamada das senhas e conduz o atendimento conforme a ordem definida.

### 🔢 Controle de Senhas
Permite visualizar, organizar e acompanhar as senhas geradas.

### ❌ Remoção de Senhas
Possibilita excluir senhas do sistema quando necessário.

---

## 🏦 Serviços Bancários

- 🆕 **Abertura de Conta**  
- 🔒 **Encerramento de Conta**  
- 💸 **Transferências**  
- 💰 **Consulta de Saldo**

---

## 🧠 Funcionamento

O sistema utiliza **estruturas de dados** para organizar as senhas, garantindo um fluxo eficiente de atendimento.

Além disso:

- Os dados podem ser **armazenados em arquivos**, permitindo persistência das informações  
- São aplicados métodos de **busca e ordenação** para gerenciamento das senhas  
- O sistema também implementa **recursividade** em partes específicas da lógica  

---

## ⚠️ Tratamento de Erros

A aplicação inclui validações para evitar falhas durante o uso, como:

- Entradas inválidas do usuário  
- Operações inexistentes  
- Tentativas de acesso a dados não disponíveis  

---

## 🚀 Objetivo do Sistema

Fornecer uma simulação simples e eficiente de um sistema bancário, organizando o atendimento de clientes por meio de senhas e permitindo a execução de operações básicas de forma controlada.
